This project is made by group 5 and the authors of this project are Djura Beenen, ![Daan Boitelle]https://github.com/daanfhb, Marit van Wolffelaar, Lisa Franse and Jord Stubb√©.

The dataset used originates from CBS StatLine and provides detailed statistics on mobility patterns in the Netherlands. It can be accessed in the CBS StatLine database through the following link: https://opendata.cbs.nl/statline/#/CBS/en/dataset/84710ENG/table?ts=1759132886441

The following filters were applied to the dataset:
- Population: Population 6 years or older
- Margins: Value
- Travel motives: All
- Travel modes: All
- Periods: 2018, 2019, 2020, 2021, 2022, and 2023
- Region characteristics: The Netherlands & Provinces
- Topic: Average per person per year

This repository contains:
- The final report as Jupyter Notebook: final_report_python_project_group5.ipynb
- The final report as HTML: final_report_python_project_group5.html
- The data used: data folder
    - Raw data: per_person__travel_modes__travel_purpose_08102025_094303.csv
    - Cleaned data: clean_data.csv
- Code for cleaning the data: data_cleanser.ipynb
- Code for the subquestions: subquestions folder
- The figures used: figures folder
