# TIL6022 Project - Group 5
![MADE WITH JUPYTER](https://img.shields.io/badge/Made%20with-JUPYTER-orange?style=flat&logo=jupyter)

This is a project for the TIL Python Programming course at the TU Delft. 
This project is made by group 5 and the authors of this project are [Djura Beenen](https://github.com/djurabeenen), [Daan Boitelle](https://github.com/daanfhb), [Marit van Wolffelaar](https://github.com/maritvw), [Lisa Franse](https://github.com/LisaFranse) and [Jord Stubb√©](https://github.com/jordstub).

The dataset used originates from CBS StatLine and provides detailed statistics on mobility patterns in the Netherlands. It can be accessed in the CBS StatLine database through the following link: https://opendata.cbs.nl/statline/#/CBS/en/dataset/84710ENG/table?ts=1759132886441

The following filters were applied to the dataset:
- Population: Population 6 years or older
- Margins: Value
- Travel motives: All
- Travel modes: All
- Periods: 2018, 2019, 2020, 2021, 2022, and 2023
- Region characteristics: The Netherlands & Provinces
- Topic: Average per person per year
